# Fördjupningsprojekt - Väderapp

## Beskrivning
Detta projekt är ett Pythonprogram som hämtar aktuell väderdata från OpenWeatherMap API.

Användaren skriver in en stad och programmet visar:
- temperatur
- väderbeskrivning
- stadens namn

## Funktioner
- API-anrop
- Felhantering
- Funktioner
- Externa moduler
- Uppdelad kodstruktur

## Struktur
- `main.py` hanterar användaren
- `weather.py` hämtar väderdata
- `requirements.txt` innehåller externa bibliotek

## Bibliotek
- requests

## Hur programmet körs
python main.py
